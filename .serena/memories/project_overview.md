# Tasks Timeline - Project Overview

## Purpose
React component library for task management and timeline visualization, published as `@tasks-timeline/components` on npm. Provides 11+ UI components, custom hooks, and utilities for building task-based applications with AI integration support.

## Key Features
- Task management with timeline visualization
- AI-powered task creation (Gemini, OpenAI, Anthropic support)
- Shadow DOM isolation for style encapsulation
- Repository pattern for storage abstraction
- Date-based task grouping and status derivation
- Comprehensive Storybook documentation

## Project Type
Dual-purpose repository:
1. **Component Library** (src/) - Published to npm
2. **Example Application** (examples/app/) - Demonstrates library usage
